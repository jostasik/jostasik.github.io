Projects

Domino's Stats

An interactive map featuring every Domino's Pizza location in the United States. Custom markers and clustering show an overview of stores at a distance and individual locations up close. Click on a store marker to see some relavent details, such as:

  • Store ID
  • Address
  • Phone Mumber
  • Business Hours
  • Delivery & Carryout Times

View the live project at: https://jostasik.com/dominos

Flowgorithm for MacOS

Flowgorithm is a popular programming tool for creating flowcharts, originally developed for Windows by Devin Cook. As a hobby project, I have created a port of Flowgorithm for MacOS, allowing Mac users to also access this valuable tool. With its user-friendly interface and extensive set of features, Flowgorithm for MacOS is a great resource for students and professionals looking to improve their programming skills and efficiency. If you're a Mac user, check out Flowgorithm for MacOS to get all the benefits of this powerful flowcharting tool.

View the project at: https://github.com/jostasik/Flowgorithm-macOS
